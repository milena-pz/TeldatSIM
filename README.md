# 📦 TeldatSIM - Gestión de Inventario de Tarjetas SIM

Este proyecto nace para resolver la complejidad en el seguimiento de stocks de tarjetas SIM y terminales móviles en entornos técnicos, proporcionando una trazabilidad clara y automatizada.

## 🎯 Contexto y Problema
En la gestión diaria de equipos de telecomunicaciones, el control manual de ICCIDs y stocks suele derivar en errores de inventario. Desarrollé esta herramienta para centralizar la información y asegurar que cada movimiento quede registrado correctamente.

## ✨ Funcionalidades Principales
- **Control de Entradas/Salidas:** Registro detallado de cada tarjeta SIM.
- **Validación de Datos:** Evita errores humanos en la introducción de códigos largos.
- **Búsqueda Inteligente:** Filtros rápidos para localizar equipos específicos por técnico o fecha.
